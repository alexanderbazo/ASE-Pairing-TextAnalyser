# Text Analyser

**Modus**: Die Aufgabe wird einzelnen Zweiergruppen (*Pairs*) bearbeitet

**Daten**: *Folger Digital Texts*, gesammelte Werke Shakespeares (*plays*) in Textform

## Aufgabenbeschreibung

Entwickeln Sie ein Programm, das alle Texte aus einem als Parameter übergebenen Ordner analysiert. Generieren Sie eine nach Häufigkeit sortierte Liste der dokumentenübergreifenden vorkommenden Wörter die zusätzlich auch die Anzahl der Dokumente, in denen die Worte vorkommen enthält. Das Ausgabe des Programms kann über Parameter beeinflusst werden und erfolgt in Form einer CSV-Datei mit dem Kopfzeile `Rank,Word,Frequency,Document Frequency,Document Frequency (relative)`

- `reversed`: Ausgabe der Liste in umgekehrter Reihenfolge [optional]
- `stopwords`: Datei mit Stoppwörtern, die bei der Analyse nicht berücksichtigt werden [optional]
